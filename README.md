# TorrentSearcherBot
A simple torrent searcher bot made in python


### Installation

#### The Easy Way

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/SpEcHiDe/PSonOfLars_BHMarie/tree/initial)

##### Required Variables

* `BOT_TOKEN`: Create a bot using [@BotFather](https://telegram.dog/BotFather), and get the Telegram API token.

* `START_MESSAGE`: Create your own start message
* `FOOTER_TEXT`: Create your own Footer text
